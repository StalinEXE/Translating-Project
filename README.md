# Translating Project
Данный репозиторий используется для публикации моих авторских переводов. Главная цель использования репозитория выходит из моей главной проблемы: при наличии желания переводить интересные мне проекты я не владею ни навыками программирования, ни командой, которая могла бы взять на себя техническую сторону этой работы.

Именно поэтому я выкладывую свои работы сюда для свободного распространения с возможностью принимать найденные ошибки от интересующихся и получать возможные пулл-реквесты.

Каждый перевод вы можете модифицировать, использовать в своих проектах и распространять при отдельном упоминании авторства.

На данный момент доступно для скачивания:

Текстовый перевод модификации Escape: Yavin IV - The Lost Maps для Star Wars: Jedi Academy [Бета-версия 1.0b]
